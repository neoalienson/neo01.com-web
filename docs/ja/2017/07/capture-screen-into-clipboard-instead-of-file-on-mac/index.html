Macでファイルではなくクリップボードに画面をキャプチャ

Created 2017-07-18   Updated 2025-10-22
Mac

多くの人は、フルスクリーンショットにはCommand + Shift + 3を、クロスヘア選択ツールには4を使用して、Macで画面をファイルにキャプチャする方法を知っています。しかし、画像をファイルに保存するよりも、キャプチャした画像をエディタに直接貼り付けたいことが多いことに気付きました。クリップボードにスクリーンショットを撮るためのホットキーは非常に不便で、Control + Command + Shift + 3または4です。以下の手順は、ファイルではなくクリップボードにスクリーンショットを撮るためのショートカットを再割り当てする方法を示しています。

1. システム環境設定に移動し、キーボードをクリックします。

2. 2番目のキーをクリックして、「画面の画像をクリップボードにコピー」にCommand + Shift + 3を再割り当てします。後で解決されるため、競合アイコンが表示されても問題ありません。

3. 最初のキーをクリックして、あまり頻繁に使用しないControl + Command + Shift + 3を再割り当てします。

4. クロスヘア選択ツールでキャプチャする場合も同じことを行います。ファイルにスクリーンショットを撮るための私の個人的な好みはOption + Command + Shift + 3です。
