これはパート1の続きです。

ゲーム

これは厳密にはアートスタイルではありませんが、さまざまなビデオゲームのグラフィックスは画像生成に大きな影響を与えます。

スターデューバレー

プロンプト: PC Game Stardew Valley style. a lone llama in an oasis inside a wild desert.

このゲームのレトロな美学は、鮮やかな8ビット風のアートスタイルを特徴とし、大胆で堅実なピクセルと明確なエッジを持っています。

ホロウナイト

プロンプト: PC Game Hollow Knight style. a lone llama in an oasis inside a wild desert.

このゲームのアートスタイルは際立っています。神秘的で古代の森の幻想的に美しいイラストで、骨のような指のように月に向かって伸びる節くれだった木の枝が、不気味な霧と光る菌類の成長のハローに囲まれています。

サイバーパンク

プロンプト: Cyberpunk style. a llama in an oasis

暗く、ネオンに浸されたサイバーパンクの風景とキャラクターは、ざらざらした工業的なテクスチャと脈打つ回路を持ち、伝統的なSF要素とざらざらした都市の荒廃を融合させています。

マインクラフト

プロンプト: Minecraft style. a llama in an oasis

マインクラフトのブロック状のローポリ美学は、シンプルで分厚い3Dモデルと鮮やかなテクスチャを特徴とし、アースカラーと微妙な照明効果に焦点を当て、幻想的な世界に命を吹き込みます。

テトリス

プロンプト: Tetris style. a llama in an oasis

テトリスの象徴的なブロックベースの美学は、幾何学的な形状とクリーンなラインを持つシンプルなモノクロームの配色を特徴とし、時の試練に耐えたミニマリストでありながら魅惑的なビジュアルアイデンティティを作り出しています。

デジタル

ピクセルアート

プロンプト: Pixel art style. a llama in an oasis

この画像は、初期のビデオゲームを彷彿とさせるレトロなデジタル美学を提供します。

映画的技法

鳥瞰図

プロンプト: Bird’s-Eye View. a lone llama in an oasis inside a wild desert.

鮮やかなビジュアルは、鳥の視点から豊かで風変わりな世界を描き、複雑なディテールと柔らかな陰影が没入感のある体験を生み出します。

クローズアップ

プロンプト: Close-Up. a lone llama in an oasis inside a wild desert.

ディテールにズームインする写真スタイルで、複雑なテクスチャ、微妙な表情、細かな特徴を印象的な親密さで捉えます。

頭部クローズアップ 大きな鼻 魚眼ショット

プロンプト: Head Close-Up Big Nose Fisheye Shot. a lone llama in an oasis inside a wild desert.

間近で撮影された魚眼ショットのポートレートは、大きな鼻と愛らしい画像を生み出し、可愛らしさと親密さを醸し出します。

アナモルフィック

プロンプト: Anamorphic style. a lone llama in an oasis inside a wild desert.

画像を水平方向に圧縮する映画技法で、誇張されたエッジと奥行きの手がかりを持つユニークで歪んだ視点を作り出し、映画やテレビ番組でノスタルジックまたはレトロな美学を作り出すためによく使用されます。
