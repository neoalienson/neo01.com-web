3Dプリント製スイッチブロッカー Cherry MX

Created 2023-05-01   Updated 2024-04-29
3D printing

キーボードで間違ったキーを誤って押してしまうことにうんざりしていませんか？不正なCaps LockやWindowsキーのせいで貴重な時間を失っていませんか？それとも私のように、マッピング後に実際に必要以上のキーを持つキーボードを使用していて、不要な雑然さを抱えていませんか？心配しないでください、友よ、このあまりにも一般的な問題には解決策があります：Cherry MXスイッチ用の3Dプリント製スイッチブロッカーです！

スイッチブロッカーのサイズは18.8x18.8mmの完璧な正方形で、高さは1mmです。ただし、LEDライトを遮ることを心配している場合は、厚さを増やす必要があるかもしれません。しかし心配はいりません。デザインを簡単に修正して、あなたに合うようにできるからです。

しかし、このスイッチブロッカーを本当に際立たせているのは、すべての側面にある切り欠きです。これにより、スイッチブロッカーが確実に所定の位置に留まり、最も不便な時に飛び出すことがありません。だから、ハイステークスなゲームの最中でも重要な仕事のプロジェクトでも、スイッチブロッカーが所定の位置に留まることを安心できます。

そして最高の部分は？無料の3DモデリングウェブサイトであるTinkercadからこのスイッチブロッカーのモデルをダウンロードできることです。だから3Dモデリングスキルがなくても、独自のカスタムスイッチブロッカーを作成できます！そしてコストを心配している場合は、心配しないでください。印刷コストを節約するために多くのスイッチブロッカーを一緒にリンクできます。実際、タオバオで注文すると、中国本土から香港への送料を含めて約10米ドルで40個のスイッチブロッカーを入手できます。

では、何を待っているのですか？イライラする誤ったキー押下にさよならを言い、Cherry MXスイッチ用の独自の3Dプリント製スイッチブロッカーで、より効率的で楽しいタイピングやゲーム体験にこんにちはと言いましょう。ハッピープリンティング！
