ツール

開発者とコンテンツクリエイター向けの便利なツールの厳選コレクションです

📝 テキスト処理

📊 テキスト統計

テキストを分析し、文字数、単語数、LLMトークン推定を含む包括的な統計を取得します。コンテンツ作成とAPI計画に最適です。

  • 8種類のテキストメトリクス
  • LLMトークン推定
  • 読書時間計算

🔤 ケース変換

camelCase、snake_case、kebab-caseなど、さまざまなケース形式間でテキストを変換します。異なる命名規則を扱う開発者に必須です。

  • 15種類のケース形式
  • 簡単に比較できるグリッドレイアウト

🎨 ASCIIテキストドロワー

さまざまなフォントを使用してテキストをASCIIアートに変換します。ドキュメントやターミナルアプリケーション用のバナー、ヘッダー、装飾テキストの作成に最適です。

  • 複数のASCIIアートフォント
  • クリーンでレスポンシブなインターフェース

📻 NATO音声記号変換

明確なコミュニケーションのためにテキストをNATO音声記号に変換します。無線通信、情報のスペルアウト、軍事/航空コンテキストに必須です。

  • 6種類の出力形式
  • 文字、数字、特殊文字に対応

😀 絵文字ピッカー

検索とカテゴリフィルタリングで絵文字を閲覧します。ソーシャルメディア、メッセージング、コンテンツ作成に最適です。

  • カテゴリベースの閲覧
  • 絵文字名で検索
  • 全カテゴリで500以上の絵文字

🔧 エンコーディングとフォーマット

🔐 Base64エンコーダー/デコーダー

オプションのURLセーフモードでBase64文字列をエンコードおよびデコードします。データエンコーディング、API統合、安全なデータ送信に必須です。

  • 標準およびURLセーフBase64エンコーディング
  • 双方向変換（エンコード/デコード）
  • Unicodeと特殊文字のサポート
  • 無効な入力のエラー処理

🌐 URLエンコーダー/デコーダー

インターネット上で安全に送信するためにURLとテキストをエンコードおよびデコードします。Web開発とAPI作業に必須です。

  • 手動エンコード/デコード制御
  • 無効な入力のエラー処理

📋 JSONバリデーター＆フォーマッター

詳細なエラーメッセージでJSONを検証、フォーマット、圧縮します。垂直レイアウトと高度なフォーマットオプションを備えた完全なJSONツールです。

  • エラー詳細付き検証
  • キーをアルファベット順にソートするオプション
  • カスタマイズ可能なインデント（2スペース、4スペース、タブ）
  • フォーマットと圧縮操作

🔄 JSON to YAML変換

双方向変換でJSONとYAML形式間で変換します。設定ファイルと異なるシステム間のデータ交換に必須です。

  • 双方向変換（JSON ↔ YAML）
  • プリティフォーマットオプション
  • 無効な入力のエラー処理

🔒 セキュリティと分析

🛡️ Defang ツール

セキュリティレポートとドキュメントで安全に共有するために、URL、IP、電子メールアドレスを defang および refang します。悪意のあるリンクの誤クリックを防ぎます。

  • URL、IP、電子メールを defang
  • 双方向変換（defang/refang）
  • セキュリティ分析に必須

🔒 ハッシュテキスト

さまざまなアルゴリズムを使用して任意のテキストの暗号化ハッシュを生成します。データ整合性検証、パスワードハッシュ、セキュリティアプリケーションに便利です。

  • 7つのハッシュアルゴリズム（MD5、SHA1、SHA256、SHA512、SHA3-256、SHA3-512、RIPEMD160）
  • 安全なクライアント側処理

🔑 X.509証明書デコーダー

PEM形式のX.509証明書をデコードおよび分析します。証明書の詳細、有効期間、セキュリティ情報を抽出します。

  • PEM証明書解析
  • 証明書フィールド抽出
  • セキュリティ情報表示

🔍 URLパーサー

プロトコル、ホスト、パス、クエリパラメータ、フラグメントを含むURLコンポーネントを解析および分析します。デバッグとURL分析に最適です。

  • すべてのURLコンポーネントを抽出
  • クエリパラメータテーブルビュー

⚙️ システム管理

🔧 Chmod計算機

8進数と記号表記でUnixファイルパーミッションを計算します。システム管理とファイルセキュリティに必須です。

  • インタラクティブなパーミッションチェックボックス
  • 8進数と記号表記の表示
  • 双方向変換
  • 一般的なパーミッションの例

⏰ Crontabジェネレーター

ビジュアルインターフェースと人間が読める説明でcron式を生成します。タスクのスケジューリングと自動化に最適です。

  • ビジュアルドロップダウンインターフェース
  • 人間にわかりやすい説明
  • 一般的なスケジューリングパターン

🌐 IPv4サブネット計算機

包括的なネットワーク詳細を含むCIDR表記からIPv4サブネット情報を計算します。ネットワーク管理と計画に必須です。

  • 完全なサブネット情報
  • ネットワークとブロードキャストアドレス
  • ホスト範囲とカウント計算
  • バイナリサブネットマスク表示

🛠️ ユーティリティ

🌡️ 温度変換

摂氏、華氏、ケルビン、ランキンスケール間で温度を変換します。科学計算と国際コミュニケーションに必須です。

  • 4つの温度スケール（°C、°F、K、°R）
  • インタラクティブなスケール切り替え

⏱️ クロノメーター

ラップ機能付きの精密ストップウォッチとタイマーです。アクティビティ、ワークアウト、または時間に敏感なタスクのタイミングに最適です。

  • ミリ秒精度のタイミング
  • 開始、一時停止、停止、リセット制御
  • 個別の期間を含むラップタイム記録
  • モバイル使用のためのレスポンシブデザイン

🌍 世界時計

複数のタイムゾーンの時計を同時に表示します。タイムゾーン間の調整、国際会議のスケジューリング、またはグローバル時間の追跡に最適です。

  • 更新付き複数タイムゾーン表示
  • 簡単な選択でタイムゾーンを追加/削除
  • 日付、時刻、タイムゾーン情報を表示
  • 35以上の主要都市とタイムゾーンを含む

📱 デバイス情報

デバイス、ブラウザ、画面に関する包括的な情報を表示します。高度なユーザーエージェント解析とAI機能検出を含みます。

  • 画面サイズと向きの詳細
  • ブラウザベンダーとプラットフォーム情報
  • 編集可能なユーザーエージェント解析
  • AI機能検出（Chromeの組み込みAI）

📊 Mermaid図エディター

ライブプレビューとインタラクティブコントロールでMermaid図を作成および編集します。フローチャート、シーケンス図、その他のビジュアルドキュメントの作成に最適です。

  • 更新付き図レンダリング
  • 複数の図タイプ（フローチャート、シーケンス、ガント、円、クラス）
  • 大きな図のズームとパン制御
  • クイックスタート用のサンプルテンプレート

📱 QRコード生成ツール

グラデーションカラー、複数のドットスタイル、誤り訂正レベルを備えたカスタマイズ可能なQRコードを生成します。マーケティングや共有用のブランドQRコード作成に最適です。

  • グラデーションと単色オプション
  • 6種類のドットスタイルとコーナーカスタマイズ
  • 4つの誤り訂正レベル
  • PNGとしてダウンロード

🖼️ ファイル分析

🖼️ PNGメタデータチェッカー

ドラッグアンドドロップサポートでPNGファイルからメタデータを抽出および分析します。画像プロパティ、テキストデータ、タイムスタンプ、技術情報を表示します。

  • ドラッグアンドドロップファイルアップロード
  • 完全なPNGメタデータ抽出
  • 画像プロパティと色情報
  • フォーマット済み表示のテキストデータ

📷 EXIF抽出

ドラッグアンドドロップサポートでJPEG画像からEXIFメタデータを抽出および分析します。カメラ設定、タイムスタンプ、技術的な写真情報を表示します。

  • ドラッグアンドドロップJPEGアップロード
  • 完全なEXIFデータ抽出
  • カメラ設定と露出情報
  • JPEGセグメント分析
  • ユニットテスト - 開発者ツールのJavaScriptユニットテストです。
