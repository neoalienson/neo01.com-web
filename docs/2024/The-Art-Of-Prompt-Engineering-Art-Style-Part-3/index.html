Techniques with Pens

Trois Crayon

Prompt: Trois Crayon style. a lone llama in in a oasis inside a wild desert.

The Trois Crayon drawing style is characterized by bold, expressive lines and vibrant colors.

Ink

Prompt: Ink style. a lone llama in in a oasis inside a wild desert.

Ink pen drawing is when you use liquid ink flowing from a small tip to create bold and expressive lines.

Wash and Ink Drawing

Prompt: Wash and Ink style. a lone llama in in a oasis inside a wild desert.

Washy lines with gentle ink strokes create soft, dreamy scenes that look like they’re floating on water.

Academic Drawing

Prompt: Academic Drawing style. a lone llama in in a oasis inside a wild desert.

Academic Drawing is a style that uses precise lines and careful shading to show what something looks like in a very realistic way.

Doodling

Prompt: Doodling style. a lone llama in in a oasis inside a wild desert.

“Doodling” means making quick and playful drawings with simple lines and shapes that are easy to make and don’t need to be perfect. Sometimes it becomes perfect.

Pencil Drawing

Prompt: Pencil Drawing style. a lone llama in in a oasis inside a wild desert.

Pencil drawing is a way to create art using thin lines and soft marks made with a pencil, giving the picture a gentle and delicate look.

Simple Pen Drawing

Prompt: Simple Pen Drawing style. a lone llama in in a oasis inside a wild desert.

Simple pencil drawings has less shade and soft marks.

Simple Color Pencil Drawing

Prompt: Simple Color Pencil Drawing style. a lone llama in in a oasis inside a wild desert.

Color Pencil drawings is a way to create art using thin lines and soft marks made with color pencils, giving the picture a gentle and delicate look.

Sketching

Prompt: Sketching style. a lone llama in in a oasis inside a wild desert.

Sketching is a way to draw quick and simple pictures using soft lines and gentle strokes that help bring out the main shapes and features.

Line Art

Prompt: Line Art style. a lone llama in in a oasis inside a wild desert.

Line art is the process of making an outline.

Line Art Contour Drawing

Prompt: Line Art Contour Drawing style. a lone llama in in a oasis inside a wild desert.

Line Art Contour Drawing: A style where bold lines and shapes show the edges and outlines of subjects in layers."
