如何在 iOS 上为 MITM 代理设置根证书

Created 2018-03-28   Updated 2025-10-18
iOS Test Automation

在 iOS 上信任调试代理（如 BrowserMob-Proxy）的根证书在其前进方向上相当严格。您可以使用设备的 Safari 从 ca-certificate-rsa.cer 下载证书，或者您也可以将文件拖曳到模拟器中。

  1. 点击允许以安装证书

  2. 点击右上角的 Install

  3. 再次点击右上角的 Install

  4. Install

  5. 验证后，点击 Done。证书已安装

  6. 要将证书信任为根证书，请前往 General 中的 About

  7. 向下滚动直到看到 Certificate Trust Settings

  8. 切换 LittleProxy MITM 以信任它

  9. 点击 Continue 以将其信任为根证书

  10. 完成。现在所有流量都可以被代理拦截而不会有任何抱怨
