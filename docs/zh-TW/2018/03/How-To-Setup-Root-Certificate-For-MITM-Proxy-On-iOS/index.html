如何在 iOS 上為 MITM 代理設定根憑證

Created 2018-03-28   Updated 2026-03-08
iOS Test Automation

在 iOS 上信任除錯代理（如 BrowserMob-Proxy）的根憑證在其前進方向上相當嚴格。您可以使用裝置的 Safari 從 ca-certificate-rsa.cer 下載憑證，或者您也可以將檔案拖曳到模擬器中。

  1. 點擊允許以安裝憑證

  2. 點擊右上角的 Install

  3. 再次點擊右上角的 Install

  4. Install

  5. 驗證後，點擊 Done。憑證已安裝

  6. 要將憑證信任為根憑證，請前往 General 中的 About

  7. 向下捲動直到看到 Certificate Trust Settings

  8. 切換 LittleProxy MITM 以信任它

  9. 點擊 Continue 以將其信任為根憑證

  10. 完成。現在所有流量都可以被代理攔截而不會有任何抱怨
