  type = string

}

function can_associate_subnet_with_vpc(subnet_id, vpc_id) {
  // perform validation logic here
}

在函式內部，您可以執行驗證變數所需的任何驗證邏輯。如果驗證成功，函式應返回 true，如果驗證失敗，則應返回 false。
