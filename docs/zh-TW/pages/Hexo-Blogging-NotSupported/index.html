巢狀格式

❌ 常見陷阱

避免這些錯誤：

  1. 硬編碼路徑

    // ❌ 不好
const path = 'C:\\Users\\neo\\file.txt';

    // ✅ 好
    
const path = require(‘path’).join(__dirname, ‘file.txt’);

  2. 缺少錯誤處理

    • 總是使用 try-catch 區塊
    • 驗證使用者輸入
    • 正確記錄錯誤

  3. 忽略 async/await

    回呼函式會導致「回呼地獄」。使用現代非同步模式。
